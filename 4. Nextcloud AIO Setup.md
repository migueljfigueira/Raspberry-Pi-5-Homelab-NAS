## 2. Initial Nextcloud Configuration

### Email (SMTP - Gmail)

* SMTP: `smtp.gmail.com`
* Port: `465`
* Encryption: `SSL/TLS`
* Login: Gmail address
* Password: Google App Password



### Fix Warnings

#### Default Phone Region

```bash
sudo docker exec -u www-data -it nextcloud-aio-nextcloud php occ config:system:set default_phone_region --value="GB"
```

#### Mimetype Migration

```bash
sudo docker exec -u www-data -it nextcloud-aio-nextcloud php occ maintenance:repair --include-expensive
```

#### Server ID (optional)

```bash
sudo docker exec -u www-data -it nextcloud-aio-nextcloud php occ config:system:set serverid --type=integer --value=1
```
