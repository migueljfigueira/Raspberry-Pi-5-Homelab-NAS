# Nextcloud AIO Backup → PC → External Drive

## 1. Create Backup (AIO)

* Open Nextcloud → **Administration settings → Nextcloud AIO**
* Set backup path:

  ```
  /mnt/backup
  ```
* Click **Create backup**



## 2. Verify Backup (Server)

```bash
ls -lah /mnt/backup
```

Expected:

```
borg/
```



## 3. Fix Permissions (Required)

```bash
sudo chmod -R o+rX /mnt/backup
```



## 4. Download to PC (Windows)

Navigate to Desktop:

```bat
cd "C:\Users\Miguel Home\Desktop"
```

Run:

```bash
scp -r miguel@SERVER_IP:/mnt/backup .
```



## 5. Move to External Drive

* Copy `backup/` folder to external drive
* Example:

  ```
  E:\nextcloud-backup\
  ```



## Notes

* Backup uses **Borg (compressed + deduplicated)**
* Do **NOT modify** contents inside `borg/`
* Do **NOT zip** (already optimized)
* Always copy the **entire folder**



## Troubleshooting

* SSH error → reboot server
* Permission denied → run `chmod` step
* SCP path issues → use `cd` + `.` method



## Result

* Full Nextcloud backup stored locally and externally
* Ready for restore and long-term storage
